### 3项目基本

这是一个中性网格策略 。上涨就开空，下跌就开多。

开始时策略会计算网格位置和数量。并且在开始的时候挂单。当挂单被成交时候，挂单新的空缺的网格单。

### 3期望行为和实际行为

当下跌开仓完成之后，价格上涨应该要能触发平仓开多的订单。但是直接跳过了了 



我感觉问题主要吃开平仓单的问题，有个简易的只关注开平仓单参数的 order.json在仓库中，可以优先查看 看看是否能定位到问题。
